## Getting starting

### Installing dependencies

- Python

```bash
pip install -r requirements.txt
```

- Node

```bash
cd frontend
npm i
```

### Starting the Gradio server (port 7860)

```bash
cd gradio
python app.py
```

### Starting the Node server (port 5173)

```bash
cd frontend
npm run dev
```
